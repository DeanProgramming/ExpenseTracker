## Local setup

1. Ensure LocalDB (MSSQLLocalDB) is installed and running:
   - `sqllocaldb start MSSQLLocalDB`

2. Add your local connection string to user secrets:
   - `dotnet user-secrets set "ConnectionStrings:DefaultConnection" "Server=(localdb)\\MSSQLLocalDB;Database=ExpenseTracker;Trusted_Connection=True;MultipleActiveResultSets=true"`

3. Apply migrations:
   - `dotnet ef database update`

4. Run the project (Visual Studio Start Debugging or `dotnet run`).
